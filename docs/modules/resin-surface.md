# 급여레진 치아면 선택

## 1. 업무 배경

### 소아 복합레진 보험청구란

소아 환자(만 12세 이하)의 충치 치료에 사용하는 광중합형 복합레진 충전은 건강보험 급여 대상이다. 청구 시 치아의 어떤 면을 충전했는지에 따라 보험코드가 달라진다.

- **U0239** — 1면 충전
- **U0240** — 2면 충전
- **U0241** — 3면 이상 충전

면수뿐 아니라 **급수(Class)**도 기록해야 한다. 급수는 충전 부위의 해부학적 위치를 분류한 것으로, 치아가 전치부인지 구치부인지, 어떤 면 조합인지에 따라 결정된다.

### GI 청구와의 차이

같은 충전 재료 청구라도 GI(글래스아이오노머)와 보험레진은 입력 방식이 다르다. GI 청구는 한 번에 20여 개 치아까지 일괄 청구가 가능하고, 면수·급수를 별도로 기재하지 않는다. 따라서 일괄입력 등의 기능이 필요하다.

반면 보험레진은 한 번에 4개 치아까지만 청구할 수 있고, 각 치아마다 면수와 급수를 명시해야 한다.

### 치아면의 구성

치아는 5개의 면을 가진다.

| 영문 | 한글 | 위치                          |
| ---- | ---- | ----------------------------- |
| B    | 협   | 볼쪽                          |
| L    | 설   | 혀쪽                          |
| M    | 근심 | 정중선에 가까운 쪽            |
| D    | 원심 | 정중선에서 먼 쪽              |
| O    | 교합 | 씹는 면 (전치부에서는 절단면) |

## 2. 실제 업무 흐름

원장이 소아 환자의 충치를 레진으로 충전한다. 진료 후 전자차트에서 해당 치아의 충전 내역을 입력해야 한다. 입력할 정보는 어느 치아의 어느 면을 충전했는지이며, 이 정보로부터 보험 청구 코드가 결정된다.

한 번의 진료에서 여러 치아를 동시에 충전하는 경우가 흔하다. 예를 들어 #46(하악 우측 제1대구치)의 O+D면과 #47(제2대구치)의 O면을 같은 진료에서 충전하면, #46은 2면(U0240), #47은 1면(U0239)으로 각각 청구해야 한다.

## 3. 문제와 설계 의도

### 이중 입력 제거

현행 덴트웹에서는 치아면(B, M, O, D, L)을 클릭한 뒤, 급수와 면수를 또다시 수동 선택해야 한다. 면을 이미 선택했으면 면수는 자명하고, 급수도 면 조합에서 자동 판별이 가능하다. 이중 입력이다.

면 선택만으로 면수와 급수를 즉시 자동 계산하도록 설계했다. 사용자가 건드릴 것은 면 토글뿐이다.

### 면수 변경 시 재입력 문제 제거

현행 시스템은 청구 버튼 선택 시점에 면수를 미리 결정해야 한다. 이후 면 입력 과정에서 면수가 바뀌면, 해당 항목을 삭제하고 처음부터 재입력해야 한다. 면수가 다른 치아를 한 번에 입력할 수도 없고, 건별로 따로 입력해야 한다.

면 선택에서 면수가 자동 계산되면, 보험코드도 면수에 맞춰 자동 연동된다. 면수가 변경되어도 코드가 따라가므로 삭제·재입력이 불필요하다. 여러 치아를 한 화면에서 동시에 입력할 수 있다.

### 급수 수동 오버라이드

급수는 대부분 자동 계산으로 충분하지만, 5급(치경부 수복)과 6급(교두정/절단연단 수복)은 면 조합만으로 판별할 수 없다. 급수 드롭다운은 항상 편집 가능하게 두되, 면을 다시 변경하면 자동 계산으로 복귀한다.

## 4. 프로토타입 설명

### 좌측: 모달 영역

고정 크기(960×680px)로 실제 모달 크기감을 전달한다.

**치식 선택기** — 상단에 FDI 표기법 기준 32개 치아를 상악/하악 좌우로 배치. 치아번호 클릭으로 추가, 다시 클릭으로 제거. 전치부(11~13, 21~23, 31~33, 41~43)와 구치부는 배경 톤으로 구분.

**면 선택 카드** — 선택된 치아마다 카드가 생성된다. 각 카드 내부:

- 치아번호 표시
- 십자형 5면 선택 도식 (상B/협, 좌M/근심, 중O/교합, 우D/원심, 하L/설). 클릭으로 토글, 선택 시 파란색
- 하단에 면수(자동) + 급수 드롭다운(자동, 수동 오버라이드 가능)

**하단 버튼** — 이전 / 저장

### 우측: 보험코드 패널

면 선택 결과가 실시간으로 반영된다.

- 치아별로 어떤 코드가 적용되는지 개별 표시 (예: #46 O+D · 2면 · 2급 → U0240)
- 코드별 횟수 합산 테이블

## 5. 정책 요약

### 자동 계산 규칙

#### 치아 위치 판별

| 구분   | 치아번호                                       |
| ------ | ---------------------------------------------- |
| 전치부 | #11~13, #21~23, #31~33, #41~43 (중절치~견치)   |
| 구치부 | #14~18, #24~28, #34~38, #44~48 (소구치~대구치) |

#### 급수 — 구치부

| 조건                         | 급수 |
| ---------------------------- | ---- |
| M, D 미포함 (O, B, L의 조합) | 1급  |
| M 또는 D 포함                | 2급  |

#### 급수 — 전치부

전치부에는 교합면이 없다. UI상 O 버튼은 그대로 두되, 내부적으로 절단면(Incisal)으로 취급.

| 조건                        | 급수 |
| --------------------------- | ---- |
| B 또는 L만 (M, D, O 미포함) | 1급  |
| M 또는 D 포함, O 미포함     | 3급  |
| M 또는 D 포함, O 포함       | 4급  |

#### 5급·6급

면 조합으로 판별 불가. 급수 드롭다운에서 사용자가 수동 선택.

### 수동 오버라이드 규칙

- 급수 드롭다운은 항상 편집 가능 (1~6급)
- 사용자가 급수를 수동 변경하면 해당 값 유지
- 이후 면 선택을 변경하면 자동 계산으로 복귀
- 면수는 선택된 면의 수이므로 오버라이드 대상 아님

### 보험코드 매핑

| 면수     | 코드  | 명칭                           |
| -------- | ----- | ------------------------------ |
| 1면      | U0239 | 광중합형 복합레진 충전-1면     |
| 2면      | U0240 | 광중합형 복합레진 충전-2면     |
| 3면 이상 | U0241 | 광중합형 복합레진 충전-3면이상 |

---

## 6. 기타사항

- 이 모듈은 소아 레진 보험청구에 특화되어 있으나, 면 선택 + 급수 자동 계산 패턴은 다른 보험 충전 항목에도 동일하게 적용 가능.
- 치식 선택기의 FDI 배치: 상악 18~11 | 21~28, 하악 48~41 | 31~38.
